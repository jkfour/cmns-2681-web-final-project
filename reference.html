Html Features

Structural tags are used through the website. Each page has a header containing an unordered list, a main, and a footer block. The main block is divided into sections where appropriate, with the home page having content separated into an article and aside. Other notable tags are blockquote at the bottom of the page, orderd list encompassing the code of conduct, and description list encompassing what the business offers.

Inline images, both static (the cookies at the bottom of the page) and clickable (the phone and email symbols in the contact block)

Special characters include © in the footer of each page, and ❮ and ❯ used as arrows to navigate the event slideshow on the index page

Meta tags used for charset, description, keywords, author and viewport

CSS

Specific contextual selectors: e > f used on hours and contact info, e + f used on the first paragraph. Attribute selectors used on links to social media accounts in the footers, as well as the submit and reset buttons on the contact page

Dynamic pseudo-classes used on all text links, :focus used on the form elements, first of type and last of type used on the event slideshow info paragraphs at the bottom of the slides.

Borders and rounded corners used extensively, background-image and size on the menu page, along with background-clip and origin on the menu image borders.

Drop-cap.

Layout tecniques: Grid:Menu & Our Space & Products. Flex: Index. Float: Code of Conduct. Positioning: Fixed positioning used on Menu images that are expanded when clicked. Absolute position used on the event Slideshow card headers and footers.

Text shadow used on website title in header, box shadow used on navigation links and containers throughout the website.

Transform rotate feature used on website logo image in cross-page header

Opacity applied to rooks at the bottom of the page, and the playing cards at the bottom of the page.

Web Form

The Contact page holds two styled forms, one for sending the business a message and the other for sending an informal application to work for the business. Both are style and use inline validation for required elements that shows both what is required to be filled and whether it is formatted correctly.

Multimedia

Stock video and audio can be found on the Our Space page. Animation are used on the gradient borders around the Menu images as well as the slideshow on the Home page which also has a fade-in effect on changing slides. The slideshow also has transitions used on the ❮ and ❯ used for navigation of the slideshow.

Javascript Usage

  1. A handler for the mobile navigation bar that resets the open state when the screen is resized to a desktop width
  2. A Slideshow container that holds slides of regularly scheduled events that can be navigated with either the arrows or bubbles underneath. The slideshow also transitions on an interval.
  3. A calendar that populates the current month with the correct layout and indicates days of regularly hosted events from an array of event objects.
  4. A Product release countdown that displays the release date along with a dynamic countdown that utilizes system date.
  5. The Menu page has clickable images that are expanded to cover the screen for closer viewing as well as adding an overlay on the page to blur the background

Most of the site is designed with at least two media queries to separate mobile and desktop screen sizes, with page layouts adjusting accordingly, however the Menu page required a third query in between to solve for a better layout.
